# 积分世界（多端同步版）

本项目使用 Supabase（Auth + Postgres + Storage）实现家里多台设备的数据同步。

## 本地运行

1. 安装依赖

```bash
npm i
```

2. 配置环境变量

复制 `.env.example` 为 `.env.local`，并填写：

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

3. 启动

```bash
npm run dev
```

## 生产构建

```bash
npm run build
```
